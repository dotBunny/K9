# K9.OS.CopyFolder

A simple safe way to copy a folder's contents to another folder.

## Arguments

| Argument | Description                                                   | Required | Default |
|----------|---------------------------------------------------------------|----------|---------|
| `SOURCE` | The absolute path of the folder who's content will be copied. | **YES**  |         |
| `TARGET` | The absolute path of where the content should be copied.      | **YES**  |         |


## Flags

| Argument           | Description                                                                    |
|:-------------------|:-------------------------------------------------------------------------------|
| `CLEAR-TARGET`     | Should content be deleted from the TARGET folder prior to copying the content? |
