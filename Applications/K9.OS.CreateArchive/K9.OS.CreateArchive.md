# K9.OS.CreateArchive

Create an archive.

## Arguments

| Argument        | Description                                                | Required | Default |
|-----------------|------------------------------------------------------------|----------|---------|
| `SOURCE`        | The file filter to apply when adding files to the archive. |          | `*.*`   |
| `SOURCE-FOLDER` | The base folder to start adding files to the archive from. | **YES**  |         |
| `TARGET`        | The absolute path of the archive file to create.           | **YES**  |         |

## Flags

| Argument       | Description                                                             |
|:---------------|:------------------------------------------------------------------------|
| `OVERWRITE`    | If the archive file already exists, overwrite it.                       |
