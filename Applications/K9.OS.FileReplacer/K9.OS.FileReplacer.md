# K9.OS.FileReplacer

A tool for replacing content in a file in one-shot.

## Arguments

| Argument | Description | Required | Default  |
| --- | --- | --- | --- |
| `SOURCE` | The absolute path of the file to read the content to be replaced from. | **YES** | |
| `TARGET` | The absolute path of the file to write the replaced content to. | No | `SOURCE` |
| `KEY` | A comma delimited list of strings to replace. | **YES** | |
| `VALUE` | A comma delimited list of values to replace the identified keys with. | **YES** | |

## Notes

The ability to have multiple `KEY` and `VALUE` arguments is useful when trying to do a single-shot replacement of a configuration file before it is used.

```
$ K9.OS.FileReplacer.exe ---SOURCE=/my/file ---TARGET=/my/working/directory/file ---KEY=__PLATFORM__,__CONFIGURATION__ --VALUE=Win64,Shipping
```
