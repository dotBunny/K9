# K9.Publish.GitHubCommitStatus

An application to update a specific GitHub commit status message.

## Arguments

| Argument      | Description                                                                         | Required | Default    |
|---------------|-------------------------------------------------------------------------------------|----------|------------|
| `AUTH-TOKEN`  | GitHub Personal Access Token - https://github.com/settings/personal-access-tokens/. | **YES**  |            |
| `REPO-OWNER`  | The repository's Owner.                                                             | **YES**  | `dotBunny` |
| `REPO-NAME`   | The absolute path of the folder to copy the file to.                                | **YES**  | `K9`       |
| `SHA`         | The full commit hash (SHA) of the target commit to update status on.                | **YES**  |            |
| `STATE`       | The state of the given description (pending, success, error, failure).              | **YES**  | `success`  |
| `DESCRIPTION` | The commit status description.                                                      |          |            |
| `CONTEXT`     | Context information about the commit status.                                        |          | `ci/k9`    |
| `URL`         | Any URL that the commit status should link to.                                      |          |            |
