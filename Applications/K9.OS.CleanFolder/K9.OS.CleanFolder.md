# K9.OS.CleanFolder

Cleans a folder and its subdirectories based on the specified exclusion filters.

## Arguments

| Argument              | Description                                                             | Required | Default |
|-----------------------|-------------------------------------------------------------------------|----------|---------|
| `TARGET`              | The absolute path of the folder to clean.                               | **YES**  |         |
| `EXCLUDE-FILE-PREFIX` | A comma delimited list of file name prefixes to exclude from the clean. | No       |         |
| `EXCLUDE-FILE-SUFFIX` | A comma delimited list of file name suffixes to exclude from the clean. | No       |         |
| `EXCLUDE-FILE`        | A comma delimited list of file names to exclude from the clean.         | No       |         |
| `EXCLUDE-FOLDER`      | A comma delimited list of folder names to exclude from the clean.       | No       |         |
| `EXCLUDE-PATH-PREFIX` | A comma delimited list of path prefixes to exclude from the clean.      | No       |         |
| `EXCLUDE-PATH-SUFFIX` | A comma delimited list of path suffixes to exclude from the clean.      | No       |         |
| `EXCLUDE-PATH`        | A comma delimited list of paths to exclude from the clean.              | No       |         |

## Flags

| Argument           | Description                                       |
|:-------------------|:--------------------------------------------------|
| `CHECK-CASE`       | Does the case of the file names and paths matter? |
| `IGNORE-EMPTY-DIR` | Should empty directories not be deleted?          |
