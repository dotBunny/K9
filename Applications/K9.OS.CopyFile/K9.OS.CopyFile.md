# K9.OS.CopyFile

Copy a file to the target folder, optionally extracting it.

## Arguments

| Argument        | Description                                          | Required | Default |
|-----------------|------------------------------------------------------|----------|---------|
| `SOURCE`        | The absolute path of the file to copy.               | **YES**  |         |
| `TARGET`        | The absolute path of the file to copy the file to.   |          |         |
| `TARGET-FOLDER` | The absolute path of the folder to copy the file to. |          |         |



## Flags

| Argument       | Description                                                             |
|:---------------|:------------------------------------------------------------------------|
| `CHECK-EXISTS` | Should we check if the target folder exists and has content?            |
| `EXTRACT`      | If the SOURCE is an archive, should we extract it to the TARGET folder? |
