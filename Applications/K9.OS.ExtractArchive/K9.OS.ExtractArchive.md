# K9.OS.ExtractArchive

Extract an archive.

## Arguments

| Argument        | Description                                | Required | Default               |
|-----------------|--------------------------------------------|----------|-----------------------|
| `SOURCE`        | The absolute path to the archive.          | **YES**  |                       |
| `TARGET-FOLDER` | The location where to extract the archive. |          | `SOURCE` folder |

## Flags

| Argument       | Description                                                             |
|:---------------|:------------------------------------------------------------------------|
| `CHECK-EXISTS` | Should we check if the target folder exists and has content?            |
