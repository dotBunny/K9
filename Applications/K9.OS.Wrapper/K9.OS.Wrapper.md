# K9.OS.Wrapper

Wrap execution of applications to control error handling and logging.

## Arguments

| Argument        | Description                                                           | Required | Default |
|-----------------|-----------------------------------------------------------------------|----------|---------|
| `COMMAND`       | The absolute path to the command/executable to be run.                | **YES**  |         |
| `ARGUMENTS`     | Arguments to be passed to the executed command.                       | No       |         |
| `REPLACE-KEY`   | A comma delimited list of strings to replace.                         | No       |         |
| `REPLACE-VALUE` | A comma delimited list of values to replace the identified keys with. | No       |         |

## Flags

| Argument           | Description                                                   |
|:-------------------|:--------------------------------------------------------------|
| `REPLACE-ERRORS`   | Replace instances of ERROR/Error/error with `[[BAD]]`.        |
| `REPLACE-WARNINGS` | Replace instances of WARNING/Warning/warning with `[[ALERT]]` |
