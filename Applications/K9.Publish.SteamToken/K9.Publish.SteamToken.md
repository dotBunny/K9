# K9.Publish.SteamToken

An application to check out and check-in the token used for SteamGuard uploads. This allows for the token to change and be updated for other agents to access.

## Arguments

| Argument           | Description                                                                                                                                         | Required | Default                     |
|--------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|----------|-----------------------------|
| `NETWORK-USERNAME` | Username for network access to be established.                                                                                                      | No       | _required for mapping_      |
| `NETWORK-PASSWORD` | Password for network access to be established.                                                                                                      | No       | _required for mapping_      |
| `NETWORK-DRIVE`    | Drive letter to map the network share to.                                                                                                           | No       | `H:`                        |
| `NETWORK-SHARE`    | Network share path to the Horde share.                                                                                                              | No       | `\\192.168.20.21\Horde`     |
| `TOKEN-TARGET`     | The absolute path to write the token file for SteamGuard to.                                                                                        | **YES**  |                             |
| `TOKEN-FOLDER`     | Where the SteamGuard tokens are stored remotely.                                                                                                    | No       | `H:\Steamworks\Tokens`      |
| `INSTALL-PACKAGE`  | The default Steam SDK to uncompress.                                                                                                                | No       | `H:\Steamworks\SDK\161.zip` |
| `INSTALL-LOCATION` | Where should the SDK be installed/extracted?                                                                                                        | No       | `D:\Steam`                  |
| `TOKEN`            | Looks to force the use of a specifically named token / user in the remote pool.                                                                     | No       |                             |
| `RETRYCOUNT`       | Sometimes, Steam randomly has some odd failures about network connectivity issues.<br />This will retry the operation a few times before giving up. | No       | `3`                         |
| `APPBUILD`         | The absolute path to the VDF file to use to facilitate the SteamCMD upload.                                                                         | **YES**  |                             |

## FLAGS

| Argument | Description |
| --- | --- |
| `FORCE` | Forces the token to be taken from the hotelling, regardless of the pre-existing lock. |
