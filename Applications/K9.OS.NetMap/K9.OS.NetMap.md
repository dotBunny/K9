# K9.OS.NetMap

Provide a mechanism for ensuring network shares are mapped as expected.

## Arguments

| Argument           | Description                                        | Required | Default                 |
|--------------------|----------------------------------------------------|----------|-------------------------|
| `NETWORK-USERNAME` | Username for network access to be established.     | **YES**  |                         |
| `NETWORK-PASSWORD` | Password for network access to be established.     | **YES**  |                         |
| `NETWORK-MAPPING`  | Drive letter / folder to map the network share to. | No       | `H:`                    |
| `NETWORK-SHARE`    | Network share path to the Horde share.             | No       | `\\192.168.20.21\Horde` |
