# K9

A command synthesizer application which searches for `*.k9.json` files in `\K9\Defaults` and recursively under `\Projects\` to build an action list.

The launched commands are run non-elevated with numerous instance environment variables set.

## Example JSON

```json
{
    "actions": [
		{
			"verb": "generate",
			"command": "GenerateProjectFiles.bat",
			"arguments": "",
			"workingDirectory": "{ROOT}",
			"description": "Generate UE5 solution file.",
			"platform": "Win64"
		},
		{
			"verb": "workspace",
			"actions":  [
				{
					"verb": "vscode",
					"command": "{LOCAL}\\Programs\\Microsoft VS Code\\Code.exe",
					"arguments": "{ROOT}",
					"workingDirectory": "{LOCAL}\\Programs\\Microsoft VS Code\\",
					"description": "Open the workspace in VSCode",
					"platform": "Win64"
				}
			]
		},
		{
			"verb": "ue5",
			"actions":  [
				{
					"verb": "editor",
					"actions":  [
						{	
							"verb": "build",
							"command": "RunUAT.bat",
							"arguments": "BuildGraph -Script=\"Engine/Build/Graph/Examples/BuildEditorAndTools.xml\" -Target=\"Tag Output Files\" -noP4",
							"workingDirectory": "{ROOT}\\Engine\\Build\\BatchFiles",
							"description": "Make a local build of the UE editor for distribution.",
							"platform": "Win64"
						},
						{
							"verb": "stage",
							"command": "RunUAT.bat",
							"arguments": "BuildGraph -Script=\"Engine/Build/Graph/Examples/BuildEditorAndTools.xml\" -Target=\"Copy To Staging Directory\" -noP4",
							"workingDirectory": "{ROOT}\\Engine\\Build\\BatchFiles",
							"description": "Copy the local UE editor build into the staging environment.",
							"platform": "Win64"
						}			
					]
				}				
			]
		}
	]
}
```
