# K9.OS.WriteTextFile

Write text content with replacements to a file.

## Auto Replaces

| Token           | With |
|-----------------|------|
| `___SPACE___`   | ` `  | 
| `___TAB___`     | `\t` |
| `___QUOTE___`   | `"`  |
| `___NEWLINE___` | `\n` |

## Arguments

| Argument  | Description                                | Required | Default        |
|-----------|--------------------------------------------|----------|----------------|
| `TARGET`  | The target file to write to.               | **YES**  |                |
| `CONTENT` | The content you wish to write to the file. | No       | `string.Empty` |
