# K9.Test.CompareImage

A tool to compare two images and fail if they are different.

## Arguments

| Argument | Description | Required | Default  |
| --- | --- |----------|----------|
| `LHS` | The absolute path of the left hand side image for comparison. | **YES** | |
| `RHS` | The absolute path of the right hand side image for comparison. | **YES** | |
| `THRESHOLD` | The percentage difference when exceeded triggers a failure. | No | `50.0f` |
| `FAILCODE` | Should the failure code be passed back to the programs exit code. | No | `false` |
