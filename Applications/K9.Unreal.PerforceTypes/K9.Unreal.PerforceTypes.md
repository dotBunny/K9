# K9.Unreal.PerforceTypes

A tool to ensure the types of files in perforce are correct.

## Arguements

| Argument           | Description                                                             | Required | Default          |
|--------------------|-------------------------------------------------------------------------|----------|------------------|
| `CHANGELIST`       | The changelist to add type changes too. This should already be created. | **YES**  |                  |
| `TARGET-DIRECTORY` | The path where to recursively look for files needing changes.           | No       | `Workspace Root` |
