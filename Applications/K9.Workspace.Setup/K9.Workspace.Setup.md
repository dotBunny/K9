# K9.Workspace.Setup

This application sets up a user's workspace just how we want it. 

- Environment path
- P4Config / P4V custom tools
- Updates and builds K9
- Execution flags on unix systems
- Add security exclusions for the workspace folder
- UE's prerequisites and version selector

> This application must be run at an elevated level; a UAC request will be triggered by the bundled manifest in the executable.
