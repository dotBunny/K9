# K9.Unreal.ToNUnit

Convert Gauntlet JSON reports to NUnit XML reports.

## Arguments

| Argument | Description                                   | Required | Default |
|----------|-----------------------------------------------|----------|---------|
| `SOURCE` | An absolute path to a Gauntlet JSON report.   | **YES**  |         |
| `TARGET` | The absolute path of the NUnit XML to output. | **YES**  |         |
| `SUITE`  | The name of the suite to convert.             | No       | `UAT`   |
